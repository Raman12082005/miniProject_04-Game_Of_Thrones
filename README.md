# Game of Thrones — Cinematic Web Experience

A scroll-driven, dark fantasy landing page inspired by the world of Game of Thrones.
Built with vanilla HTML, CSS, and GSAP ScrollTrigger.

## Features

- Video scrubbing synced to scroll position
- 6 chapter transitions with animated text
- Interactive Great Houses grid with hover reveal & 3D sigil tilt
- Loading screen, progress bar, rune bar, and corner ornaments
- Medieval aesthetic — Cinzel fonts, gold/parchment palette, grain overlays

## Tech Stack

- HTML5 / CSS3
- Vanilla JavaScript
- GSAP 3 + ScrollTrigger

## Getting Started

Clone the repo and open `index.html` in a browser.
No build step required.

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```
